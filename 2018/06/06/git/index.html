修改git目录文件的权限

将用户nobody加入git组，把/home/git/.gitolite.rc中的UMASK值0077修改为0027，把git用户设置为git-shell登录

    UMASK                           =>  0027,

其中，/usr/local/bin/git-shell为which git-shell查到的路径

git:x:1000:1001::/home/git:/usr/local/bin/git-shell
